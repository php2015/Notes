### 渲染过程

1. 模板解析成 render 函数（开发环境下完成）

2. 触发响应式 监听 data 中各属性

3. 执行 render 函数，生成 vnode，patch(elem,vnode)

### 更新过程

1. 修改 data。触发 setter（此前在 getter 中已被订阅者监听）

2. 重新执行 render 函数，生成 newVnode

3. patch(vnode,newVnode)
