# 记录项目中 一些通用的单词

```js
//一般项目结构 管理模板
├── build                      // 构建相关
├── config                     // 配置相关
├── src                        // 源代码
│   ├── api                    // 所有请求
│   ├── assets                 // 主题 字体等静态资源
│   ├── components             // 全局公用组件
│   ├── directive              // 全局指令
│   ├── filtres                // 全局 filter
│   ├── icons                  // 项目所有 svg icons
│   ├── lang                   // 国际化 language
│   ├── mock                   // 项目 mock 模拟数据
│   ├── router                 // 路由
│   ├── store                  // 全局 store 管理
│   ├── styles                 // 全局样式
│   ├── utils                  // 全局公用方法
│   ├── vendor                 // 公用 vendor
│   ├── views                  // 页面
│   ├── App.vue                // 入口页面
│   ├── main.js                // 入口 加载组件 初始化等
│   └── permission.js          // 权限管理
├── static                     // 第三方不打包资源
│   └── Tinymce                // 富文本
├── .babelrc                   // babel-loader 配置
├── eslintrc.js                // eslint 配置项
├── .gitignore                 // git 忽略项
├── favicon.ico                // favicon 图标
├── index.html                 // html 模板
└── package.json               // package.json
```

```js
alias               // 别名
redirect            // 重定向
snippets            // 片段
permission          // 准许
include             // 包括;包含;使成为…的一部分
proxy_pass          // 代理通行证
sendfile            // 发送文件
Cache-Control       // 高速缓冲存储器-客户端
certs               // 证书
```
